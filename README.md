# 📋 Attendance Management System

A Django-powered web application to manage student attendance, subjects and check attendance.

---



## 🧩 Features

- **Admin**:
  - Manage staff, students, courses and attendance reports
  - add students, take attendance and show attendance

---


## 🛠️ Tech Stack

- **Backend**: Django (ORM, Views, Admin)
- **Database**: SQLite (default development); can be switched to PostgreSQL/MySQL
- **Frontend**: Django templating with Bootstrap (or your preference)
- **Media**: Uses `media/voice_notes`, `media/temp`
- **Environment management**: `venv`, `.env`

---

## ⚙️ Prerequisites

- Python 3.10+
- `virtualenv`
- (Optional) Git
